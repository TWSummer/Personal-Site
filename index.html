Friendr

The place for making friends

Live Site

Friendr is an OKCupid inspired site that makes it easy for people to make new friends!


The site is built on a Ruby on Rails backend and a React/Redux frontend. Features include:


  • Search: The right friend for you is not the same as for everyone else. You likely care about qualities like age, location, and whether they answered questions similarly to yours. Friendr's search tool makes this process easy and efficient.
  • Profile: Show off what makes you, you! Upload a photo and write a bit about yourself, then utilize the integration with Google Maps API to set your location so people looking for you know that you are in their area!
